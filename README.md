# Pizzaria da Nonna

Este site foi desenvolvido como projeto final na disciplina de "Desenvolvimento web", ele consiste no site de uma pizzaria ficticia chamada "Pizzaria da Nonna"

## Objetivo

O projeto precisava ser desenvolvido utilizando HTML, CSS e JavaScipt. Ele precisava utilizar todos os conceitos estudados em aula, sendo posteriormente aprovado pelo professor.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Funcionalidades

- Página inicial
- Cardápio
- Página de contato
- Política de privacidade

## Status

Versão 1.0 para entrega acadêmica
